## Description
The auth service based on access and refresh tokens with the email confirmation.

## Tech stack
 - TypeScript
 - NestJS
 - PostgreSQL
 - jwt
 - Prisma
 - Node.js v22.11.0

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
